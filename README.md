## Description
Little Pet project (Blog system) to learn [TypeScript](https://www.typescriptlang.org/),
[Nest](https://github.com/nestjs/nest) framework and [React](https://reactjs.org)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
